= detail 컴포넌트에 직접 주소를 입력하여 들어가면 props가 전달이 안 됨.
그 때 강제로 Home 컴포넌트로 돌려보내는 것이 리다이렉트.

Ajax(Asynchronous Javascript and XML)은 웹페이지의 이동 없이 현재 페이지에서 필요한 데이터만 로드 또는 전송하는 기술을 말한다.
보통의 경우 데이터 처리는 요청받은 순서대로 진행하지만 AJAX는 순차적으로 진행하지 않는다. 
이러한 방식을 '비동기식 방식'이라고 부른다.

JSON은 자바스크립트 객체에 리터럴 (키-값) 형식의 구문으로 구성된 데이터이다.
JSON은 외부로 전송하여 데이터를 처리할 수도 있고 내부로 로드하여 처리할 수도 있다.

$.ajax()메소드를 이용하여 외부에 있는 JSON파일을 로드하여 처리하는 방법 실습.

json, xml, html

xhtml - xml과 html을 합친 것. 

xml (eXtensible Markup Language) 태그의 목적은 해당 태그가 가지고 있는 데이터의 종류를 설명한다.
데이터 형식을 기술하기 위한 마크업 언어다.
작성자가 의미에 맞게 임의로 태그를 작성할 수 있다.

태그 작성법
: 대소문자 구분 / 공백 x / 첫글자는 한글, 영문, _ 사용 가능 / -, _ 태그 중간에 사용 가능.

axios는 promise 기반으로 api 형식이 다양하게 제공된다.
promise - 비동기식 처리하는 자바스크립트 객체. 데이터를 요청하고 받아올 때까지 기다렸다가 
화면에 나타내는 로직을 실행할 때 주로 사용한다.
